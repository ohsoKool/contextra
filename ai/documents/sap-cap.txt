SAP CAP stands for SAP Cloud Application Programming Model.

SAP CAP is a framework provided by SAP for building enterprise-grade cloud applications.

CAP supports Node.js and Java runtimes.

CAP uses CDS (Core Data Services) for defining data models and services.

CAP applications commonly run on SAP Business Technology Platform (SAP BTP).

CAP automatically generates OData services from CDS models.

CAP promotes clean architecture and domain-driven design.
